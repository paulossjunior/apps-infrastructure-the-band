# Apps Infraestruture

## 🚀 Goal

Provides a infrastructure to manage the data present in each OBDR/OBS. 

## ⚙️ Requirements

1. Docker Compose

## 🔧 Install

In a terminal execute the following commnad to start the infrastructure

```bash
docker-compose up
```
